The FAST package contain slightly modified feature classes from FAST Corner Detection (FAST Corner Detection).

The classes of this package should NOT be used directly except from the according classes in the de.lmu.dbs.jfeaturelib.pointDetector package.

FAST was first published By Edward Rosten in 2005, 2006:

  1. Fusing points and lines for high performance tracking (2005) and
  2. Machine learning for high-speed corner detection (2006)

Please see http://www.edwardrosten.com/work/fast.html for links to the according papers, more information and other reference implementations. At this page you can also find BSD-licensed code.
